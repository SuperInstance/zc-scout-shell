### 1. Trending Open-Source Agent Frameworks (Current Snapshot)
Based on GitHub trending and recent releases (last 30 days), the landscape shows consolidation and specialization.

*   **CrewAI** (v0.28+): Trending for its "role-based" orchestration. New focus is on **long-running agents with memory** and tool-use optimization, moving beyond simple sequential chains. This aligns with the fleet's "room" concept where agents have persistent state.
*   **LangGraph** / **LangChain**: Still dominant. The trend is toward **cyclical, stateful graphs** (StateGraph) and **persistent checkpoints**, mirroring the "Mirror Play" and iterative training seen in `plato-torch`.
*   **AutoGen** (Microsoft): Recent activity around **multi-agent debate frameworks** and **cost-optimized execution**. Less hype, more focus on research and robust multi-agent conversations.
*   **Emerging Pattern - "GPU-Native" Runtimes:** Several new projects (e.g., `sglang`, `vLLM`, `lmdeploy`) are not agent frameworks per se, but provide high-throughput, low-latency **serving runtimes** for LLMs. This is the infrastructure layer our `cudaclaw` (GPU-resident agent runtime) operates in. The trend is toward minimizing CPU-GPU data movement.

### 2. Specific Observation: The "Constraint" Pattern
A notable pattern relevant to `constraint-theory-core` is the rise of **"guided generation"** and **"grammar-based sampling"** (e.g., `outlines`, `guidance`, `lmql`). These frameworks allow LLMs to generate outputs that strictly adhere to defined schemas, grammars, or logical constraints.

*   **Fleet Relevance:** This is a **software-level implementation of geometric snapping**. Instead of snapping 3D vertices to a grid, these tools "snap" LLM token logits to a defined output space (JSON schema, regex). The core theory—defining a solution space and finding valid paths within it—is analogous. The fleet's `constraint-theory-core` (geometric) could share conceptual ground with these "constraint-based sampling" libraries (textual).

### 3. Negative Space (P0) - What's Cooling Off
*   **"Yet Another Simple Agent Framework":** Low-star repos that simply wrap `langchain` with a thin CLI are not trending. The market is saturated with basic examples.
*   **Pure Prompt-Chaining without State:** Simple sequential `llm.call()` scripts are now considered tutorials, not production frameworks. The trend demands persistent memory and cyclic graphs.

### 4. Safe Channel (P1) - Research Vector for the Fleet
The intersection of **stateful agent graphs** (CrewAI/LangGraph), **GPU-native execution** (cudaclaw's domain), and **constraint-based generation** presents a coherent research channel.

This stack would be highly efficient and deterministic.
